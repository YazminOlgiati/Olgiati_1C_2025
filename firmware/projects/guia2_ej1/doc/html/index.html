General Description

Se mide distancia con un sensor de ultrasonido: Si la distancia es menor a 10 cm, apagar todos los LEDs. Si la distancia está entre 10 y 20 cm, encender el LED_1. Si la distancia está entre 20 y 30 cm, encender el LED_2 y LED_1. Si la distancia es mayor a 30 cm, encender el LED_3, LED_2 y LED_1. Mostrar el valor de distancia en cm utilizando el display LCD. Usar TEC1 para activar y detener la medición. Usar TEC2 para mantener el resultado (“HOLD”). Refresco de medición: 1 s
